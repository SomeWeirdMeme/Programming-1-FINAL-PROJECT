PROJECT TITLE: Game Library

PURPOSE OF PROJECT:
The purpose of this project is to design and implement a Game Library system using Java.
The project allows people to manage a collection of digital games, viewing, searching,
rating, and simulating game sessions. Using things we learnt such as classes, inheritance, encapsulation, 
enumerations, and collections.

VERSION and DATE:
Version 1.0 - (April 29th 2026)

HOW TO START THIS PROJECT:a
1. Open the project in BlueJ.
2. Compile all Java classes.
3. Run the main driver class (MAIN).
4. Follow the menu instructions to interact with the project.

USER INSTRUCTIONS: 
Game Library – User Instructions

Run the program. A menu with options (0–9) will appear.
Enter the number of the thing you want and press Enter.

Option 1: Add a Game
Choose type: 1 (Single Player), 2 (Multiplayer), or 3 (Online)
Enter title, size (MB), genre (ACTION/RPG/ADVENTURE), platform (PC/PLAYSTATION/XBOX), 
and age rating (E/T/M).

Game will be added to the library.

Option 2: View All Games
Displays all games with details.

Option 3: Rate a Game
Enter the exact game title.
Enter a rating from 0 to 5.

Option 4: Filter by Genre
Enter ACTION, RPG, or ADVENTURE.

Option 5: Filter by Platform
Enter PC, PLAYSTATION, or XBOX.

Option 6: Filter by Age Rating
Enter E, T, or M.

Option 7: Add Game Session
Enter game title, date, region, and notes.
Session starts automatically.
Press Enter to end the session.
Session is saved.

Option 8: View All Sessions
Displays all recorded sessions.

Option 9: Top Rated Games
Shows highest-rated games.

Option 0: Exit
Closes the program.
