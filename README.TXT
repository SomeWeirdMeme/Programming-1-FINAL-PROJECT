PROJECT TITLE: Game Library

PURPOSE OF PROJECT:
The purpose of this project is to design and implement a Game Library system using Java.
The project allows users to manage a collection of digital games, including viewing, searching,
rating, and simulating game sessions. Using concepts such as classes, inheritance, encapsulation, 
and the use of enumerations and collections.

VERSION and DATE:
Version 1.0 - (April 29th 2026)

HOW TO START THIS PROJECT:a
1. Open the project in BlueJ.
2. Compile all Java classes.
3. Run the main driver class (GameLibrary).
4. Follow the menu instructions to interact with the project.
